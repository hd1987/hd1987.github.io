js链式取值（函数解析字符串）

发表于 2016-03-25

开发中，链式取值是非常正常的操作，如：
res.data.goods.list[0].price
有时候会出现错误：
Uncaught TypeError: Cannot read property 'goods' of undefined

通过函数解析字符串验证

使用:

var c = {
  a: {
    b: [1, 2, 3]
  }
}
get(c, 'a.b') // [1,2,3]
get(c, 'a.b[1]') // 2
get(c, 'a.d', 12) // 12
