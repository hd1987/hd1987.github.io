Github地址: [create-react-app](https://github.com/facebook/create-react-app)

安装、启动命令

```
npm install -g create-react-app
create-react-app my-app

cd my-app
npm start
```

初始目录结构

```
my-app
├── README.md
├── node_modules
├── package.json
├── .gitignore
├── public
│   └── favicon.ico
│   └── index.html
│   └── manifest.json
└── src
    └── App.css
    └── App.js
    └── App.test.js
    └── index.css
    └── index.js
    └── logo.svg
    └── registerServiceWorker.js
```
